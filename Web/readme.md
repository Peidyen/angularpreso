# Setup

## Dev Requirements
- NodeJS
- NPM ( Node Package Manager )
- Bower

## Server Requirements
- IIS URL Rewrite 2
- .NET 4.5.1

## Setup
- Install Bower `npm install -g bower`
- Install Bower Components `bower install`
- Install Node Components `npm install`

## Running
- Create a virtual directory for `Web` project in IIS and goto that URL
